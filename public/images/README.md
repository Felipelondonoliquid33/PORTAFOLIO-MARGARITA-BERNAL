# Images Folder

Place your photography images here for the PhotoGridScroll component.

## Instructions

1. Copy your 23-26 portrait images (1200px × 1600px) to this folder
2. Name them: `photo-01.jpg`, `photo-02.jpg`, `photo-03.jpg`, etc.
3. Update the `photos` array in `components/PhotoGridScroll.tsx`

## Example file names:
- photo-01.jpg
- photo-02.jpg
- photo-03.jpg
- ... (up to photo-23.jpg or more)

## Image Requirements:
- **Aspect Ratio**: 3:4 (portrait - taller than wide)
- **Size**: 1200px wide × 1600px tall (recommended)
- **Format**: JPG/JPEG or WebP
- **File Size**: Under 500KB each (optimized)
