# Public Folder - Fonts Directory

This is the `public` folder for your Next.js project.

## Font Files Location

Place your custom font files for Margarita Bernal in the `fonts/` subdirectory:

```
public/
  └── fonts/
      ├── MargaritaBernal-Regular.woff2
      ├── MargaritaBernal-Regular.woff
      ├── MargaritaBernal-Bold.woff2
      └── MargaritaBernal-Bold.woff
```

## How to Add Your Font

1. Copy your font files to `public/fonts/`
2. Open `app/globals.css`
3. Uncomment the `@font-face` declarations for "Margarita Bernal"
4. Update the file names to match your actual font files

See `HOW_TO_ADD_CUSTOM_FONT.md` in the project root for detailed instructions.
